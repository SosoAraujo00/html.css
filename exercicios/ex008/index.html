Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito ultilizando a tag <b> (não semântica).

Nesta frase temos um termo em destaque ultilizando a tag <strong> (semântica).

Itálico/Ênfase

Nesta frase temos um termo em itálico ultilizando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase ultilizando a tag <em>(semântica).

Texto marcado

Podemos criar também um texto marcado ultilizando a tag <mark> (não semântica).

E no outro parágrafo, temos outro texto marcado.

Texto grande e pequeno

Estamos criando um texto grande (ultilizando a tag <big>)e um texto pequeno (ultilizando a tag <small>) nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido (ultilizando a tag <del>) para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido (ultilizando a tag <ins>) para dar ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado ultilizando a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x2+3, ultilizando a tag <sup>

Texto subscrito

para inserir coisas do tipo H2, ultilizando a tag <sub>
